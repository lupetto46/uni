- Una successione $\{a_n\}_{n\in ℕ}$ è crescente se $a_{n+1} \geq a_n \forall n \in ℕ$
- Una successione $\{a_n\}_{n\in ℕ}$ è strettamente crescente se $a_{n+1}> a_n \forall n \in ℕ$
- Una successione $\{a_n\}_{n\in ℕ}$ è decrescente se $a_{n+1}\leq a_n\forall n \in ℕ$
- Una successione $\{a_n\}_{n\in ℕ}$ è strettamente decrescente se $a_{n+1}< a_n\forall n \in ℕ$
